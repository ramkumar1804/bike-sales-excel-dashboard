# 🚴 Bike Sales Dashboard

An interactive **Bike Sales Dashboard** built in **Microsoft Excel** to analyze customer purchasing behavior using dynamic visualizations, Pivot Tables, Pivot Charts, and Slicers. This project transforms raw sales data into actionable business insights through an intuitive and interactive dashboard.

## 📋 Project Overview

The objective of this project is to analyze bike sales data and identify purchasing trends based on customer demographics. The dashboard enables users to interactively filter data and gain insights into customer behavior across multiple dimensions.

---

## ✨ Features

- Interactive dashboard with dynamic filtering
- Pivot Tables and Pivot Charts
- Slicers for data exploration
- Professional dashboard layout
- Customer demographic analysis
- Business insight generation

---

## 📊 Dashboard Insights

- **Average Income per Purchase** by Gender
- **Customer Age Brackets** Analysis
- **Customer Commute Distance** Analysis
- Interactive filtering by:
  - Marital Status
  - Region
  - Education
  - Age Brackets

## 📈 Key Business Insights

- Customers with higher average income are more likely to purchase bikes.
- Middle-aged customers account for the highest number of bike purchases.
- Commute distance influences purchasing behavior.
- Interactive filters allow users to compare customer segments across different demographics.
